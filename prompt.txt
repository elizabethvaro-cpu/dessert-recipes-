4. API Integration

Use TheMealDB (free & open):

Base URL:
https://www.themealdb.com/api/json/v1/1/

Example Endpoints:

Desserts by category:
/filter.php?c=Dessert

Lookup recipe details:
/lookup.php?i={mealId}

Filter by area (country):
/filter.php?a={Area}

API Requirements:

Centralized API service file

Typed responses (TypeScript)

Async/await pattern

Graceful failure handling
